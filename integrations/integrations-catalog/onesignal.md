# OneSignal

Use OneSignal to send SMS from Cargo.

### How to authenticate?

In OneSignal, select **settings**, then **Keys & IDs**.

Finally, copy and paste the values of **OneSignal App ID** and **Rest API Key** in the connector config on Cargo.

### Available actions

**Write** - Send SMS
